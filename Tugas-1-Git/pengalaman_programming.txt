PENGALAMAN PROGRAMMING
pengalaman programming saya sudah dimulai dari tahun 2010, waktu itu saya nekat melamar pekerjaan ke suatu perusahaan sebagai web developer padahal saya tidak ada basic untuk programming sama sekali, disana saya belajar PHP & MySQL, mulai dari php native sampai menggunakan framework Codeigniter.
Beberapa aplikasi sudah saya buat, satu aplikasi yang membuat saya bangga membuatnya adalah ketika membuat aplikasi Proposal Online, dimana sebelum ada aplikasi ini proses pengajuan proposal masih manual, user membuat proposal di ms Word lalu di save->di print->ditanda tangan->di scan->lalu di email ke atasan untuk approval nya. Proses yang sangat ribet menurut saya, akhirnya saya membuat aplikasi Proposal Online sehingga prosesnya lebih mudah, peper less, dan lebih cepat dalam bekerja.
Hal yang paling saya senangi dalam dunia programming adalah ketika membuat satu project, lalu ada problem disana dimana kita harus solving problem tersebut, bisa melalu googling, youtube, atau forum2 di internet.
